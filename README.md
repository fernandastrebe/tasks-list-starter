## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Install the Json Server globally, command:
```
npm install -g json-server
```

### Run your fake REST API, command:
```
 json-server --delay 500 db.json
 ```

 ### Run the unit tests
 ```
 npm run test:watch
 ```
